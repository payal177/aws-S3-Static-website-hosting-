# AWS S3 Static Website Hosting

## Project Overview

This project demonstrates how to host a static website using Amazon S3. The website was deployed by creating an S3 bucket, uploading website files, enabling static website hosting, and configuring bucket permissions for public access.

## AWS Services Used

* Amazon S3
* S3 Static Website Hosting
* Bucket Policies

## Steps Performed

1. Created an Amazon S3 bucket.
2. Uploaded the website files (index.html).
3. Enabled Static Website Hosting.
4. Configured public access settings.
5. Added a bucket policy to allow public read access.
6. Accessed the website using the S3 website endpoint.

## Project Architecture

User Browser → Amazon S3 Bucket → Static Website

Successfully hosted a static website on Amazon S3 and made it accessible through a public website endpoint.
